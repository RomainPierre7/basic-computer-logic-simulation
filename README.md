# basic-computer-logic-simulation

This is a simple computer logic simulation project

## How to run
```bash
python3 main.py
```

## Logic gates

### Main gates
- NOT
- AND
- OR

### Derived gates
- NAND
- NOR
- XOR
- XNOR
